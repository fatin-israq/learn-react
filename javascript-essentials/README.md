# JavaScript Essentials for React

Essential JavaScript concepts to master before learning React.

## Topics Covered

**Core Fundamentals**

- Variables, Arrays, Strings, Objects
- Template Strings & Arrow Functions
- Spread Operator

**Array Methods**

- Map, Filter, Find, forEach

**Advanced Concepts**

- Array & Object Destructuring
- JSON & Fetch API
- Truthy/Falsy Values & Ternary Operator
- Local Storage & Session Storage
